# Trainbrain Shared
Trainbrain Shared is an npm package that contains shared code that is used in different Trainbrian applications.
For example shared types between the back-end and the front-end.
This npm package is not hosted in any npm repository. It is only used locally in Trainbrain.

## Build
This package needs to be built by running `npm run build` in the root folder of this project.
The applications that use the package don't need to do anything. The new code is instantly available to them.
